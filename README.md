# 检验科普内容工作台

一个面向医院检验科、医院宣传科和医疗公众号编辑的独立站工具，用于生成 **医院公众号 + 小红书** 双平台检验项目健康科普内容提示词。

## 产品定位

把血常规、尿常规、肝功能、肾功能、血糖血脂、肿瘤标志物、抽血空腹、报告常见误区等检验科主题，整理成患者和普通人能看懂的健康科普内容。

适合：

- 医院检验科
- 医院宣传科
- 医疗公众号编辑
- 体检中心运营人员
- 医疗新媒体代运营团队

## 功能

- 登录 / 注册
- 保存检验科普内容草稿
- 支持血常规、肿瘤标志物、抽血空腹等快速示例
- 录入检验项目、目标人群、注意事项、常见异常、误区边界
- 生成并复制双平台 AI 提示词
- 输出方向覆盖：
  - 公众号科普长文
  - 小红书健康科普笔记
  - 标题建议
  - 标签建议
  - 图文卡片分镜
  - 合规风险检查
  - 发布前审核清单
- 支持下载提示词 `.txt`

## 内容安全边界

工具只作为健康科普内容辅助，不用于：

- 个人检验报告诊断
- 治疗方案建议
- 用药建议
- 替代医生或检验科专业审核
- 承诺平台过审、涨粉、阅读量或咨询转化

发布前建议由检验科专业人员或医院宣传 / 合规部门审核。

## 本地运行

```bash
node server.js
```

默认访问：

```text
http://localhost:3000
```

## 云端部署

项目支持 Render 等 Node.js 平台：

- Start Command: `node server.js`
- Node Version: 24+
- 环境变量可选：
  - `PORT`：平台通常自动提供
  - `DATA_DIR`：数据库目录，默认 `./data`
  - `DB_PATH`：SQLite 数据库路径

注意：如果云平台没有持久化磁盘，SQLite 数据可能会在服务重启后丢失。正式使用建议绑定持久化磁盘或改成云数据库。
